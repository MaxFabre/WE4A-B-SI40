# Front-End

## Requirements
Utiliser ces commandes afin d'obtenir les packages du front-end:
(vous pouvez le faire directement depuis le terminal de votre IDE)  
`composer require twig`  
`composer require symfony/asset`

## Resources
Les resources (CSS, images et autres) sont à placer dans le dossier public (celui qui est chargé par le client).
